# 实现计划

## 概述
本实现计划将静态HTML即将推出页面转换为Next.js组件，同时保持所有功能和视觉设计。

- [x] 1. 创建Next.js页面组件结构


  - 创建 `app/hello-preview/page.tsx` 或 `app/coming-soon/page.tsx`
  - 设置基础组件结构和TypeScript类型定义
  - 配置页面元数据（标题、描述）
  - _需求: 1.1, 1.2, 1.3, 1.4, 1.5_



- [ ] 2. 实现导航栏组件
- [ ] 2.1 创建带Logo和首页按钮的导航栏组件

  - 使用Next.js Image组件实现Logo图片
  - 添加首页图标SVG按钮
  - 应用玻璃态样式（rgba背景 + backdrop-filter）

  - _需求: 1.1, 4.1, 4.3, 4.4_

- [x] 2.2 添加导航栏交互效果

  - 实现Logo悬停动画（放大1.08倍 + 旋转5度）

  - 添加导航链接和正确的路由
  - _需求: 4.6, 4.8_

- [ ] 3. 构建主内容容器
- [ ] 3.1 创建即将推出卡片布局
  - 实现玻璃态卡片容器

- [ ] 4.1 创建表单结构和状态管理

  - 创建包含标签和输入框的表单JSX结构
  - 添加带正确占位符的邮箱输入框
  - 实现受控输入组件
  - _需求: 2.1, 2.2_


- [ ] 4.2 添加邮箱验证逻辑

  - 验证失败时显示错误消息
  - 错误时将输入框边框颜色改为红色

  - 添加加载旋转器动画（spin关键帧）
